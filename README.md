# 🚀 AI Career Copilot

AI Career Copilot is an AI-powered career assistant built using Python, Streamlit, and Google's Gemini AI.

The application helps job seekers analyze their resumes against job descriptions, identify skill gaps, generate interview questions, create personalized cover letters, and draft recruiter outreach messages.

## Features

### 📊 ATS Score Analysis

* Resume vs Job Description matching
* ATS compatibility score
* Skill match percentage

### ✅ Skill Gap Detection

* Identifies matched skills
* Highlights missing skills
* Recommends areas for improvement

### 🤖 AI Recruiter Review

* Resume strengths analysis
* Resume weaknesses analysis
* ATS improvement suggestions
* Interview readiness assessment

### 🎤 AI Interview Question Generator

* Technical questions
* Project-based questions
* Behavioral questions
* Personalized according to resume and JD

### 📄 Cover Letter Generator

* Creates professional cover letters
* Tailored to specific job descriptions

### 📧 Recruiter Outreach Message Generator

* Generates LinkedIn messages
* Creates recruiter email drafts
* Personalized job application outreach
